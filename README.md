# Путь клинка · квест мастера Кодзиро

Сайт-квест по Elden Ring: дух самурая мастера Кодзиро поручает собрать все катаны
Страны тростника. Квесты идут строго по порядку — следующий открывается только
после того, как ты отметил предыдущий клинок собранным.

## Запуск

```bash
npm install
npm run dev      # дев-сервер
npm run build    # прод-сборка в dist/
```

## Цепочка квестов

1. **Утигатана** (стартовая / Помертвелые катакомбы)
2. **Длинный клык** (линия Юры)
3. **Комплект из Страны тростника** — облачение, открывается после 2-й катаны
4. **Меч из метеоритной руды** (Звёздные пустоши)
5. **Лунная вуаль** (Туннель Гаэля, Магмовый змей)
6. **Меч из драконьей чешуи** (Озеро Гнили, Драконий солдат)
7. **Финал** — последний дар: спящий дракон, добиваемый кровотечением

## Особенности

- Прогресс хранится в `localStorage` (кнопка «Начать заново» сбрасывает).
- Защита от мисклика: отметить клинок собранным можно лишь через двухступенчатое
  подтверждение («Да, клинок у меня» → «Скрепить обет»).
- Диалоги мастера — в стиле Elden Ring (печатающаяся анимация, портрет).
- Все картинки — плейсхолдеры. Подставь свои по инструкции в
  `public/images/README.md` (имена файлов уже прописаны).

## Где править

- Тексты, лор, локации, диалоги, порядок квестов → `src/data/quests.js`
- Пути к картинкам → там же, поля `image` / `mapImage` / `portrait`
- Стиль / цвета → `tailwind.config.js`, `src/index.css`
